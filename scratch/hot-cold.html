Hot or Cold: Mystery Number

Can you guess the mystery number?!? It's between 0 and 100.

1) Choose a backdrop and a character to host the game.

2) Have your host intoduce the game and explain what this game is about. "I have a mystery number between 0 and 100. Can you guess what it is?" Remember to make sure the player has a space to enter their guess and start the playing.

3) It's time to make the mystery number!

When the player enters a guess, have your host check if the answer is a number between 0 and 100. If it isn't remind the player of the rules.

Then check if the answer is the same as as the mystery number. If the guess is equal to the mystery number, celebrate! Else, the host should ask the player to try again.

4) Let's keep track of the player's guesses by creating a list of guesses. Add each new answer to the list.

If the player repeats a guess, let them know they've guessed it already. Else add the answer to list and tell them they didn't get it right.

Make it so that the list is visible only after the player guesses.

5) Can we make the host tell the player how many tries it took for them to guess the mystery number? We can use the length of the list to construct the response. ;-)

6) It's time to give our player hints of how far their guess is from the mystery number. Let's start by calcualting the difference between the guess and the mystery number.

7) Let's start giving the player feedback about their guesses.

If the difference between the guess and the mystery number is less than 5, then the guess is "HOT!"; else "Cold"

If the difference is between 5 and 10, they're guess is "WARM!"; else "COLD!", etc..

You can add as many layers as you'd like. We added Hot, Warm, Luke Warm, Tepid, Cold, ICE-Cold.

Hint: use the absolute value of the difference to measure how far the guess is. The block you're looking for is in operators.

8) Let's clean up our code. The player should never see the mystery number and the difference. The list should be hidden until someone makes a guess.
