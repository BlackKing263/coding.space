Neon Flag

In this project you'll make your first web page!

For an example, click here

  1. Give you page a title, it can be whatever you'd like!
  2. Using HTML, make at least 5 divs in the body of the page.
  3. Give each div a unique ID.
  4. Using CSS, set the size of the divs. Make them 100px tall, and have them stretch across the whole page.
  5. Set the background color of each div. You can make a rainbow, choose your favorite colors, or whatever you would like!
  6. Keep working on your project until you're happy with it.
  7. Submit your webpage below.
