Neon Flag

In this project you'll make your first web page!

For an example, click here

  1. Using HTML, make at least 5 divs in the body of the page.
  2. Give each div a unique ID.
  3. Using CSS, set the size of the divs. Make them 20% tall, and have them stretch across the whole page.
  4. Set the background color of each div. You can make a rainbow, choose your favorite colors, or whatever you would like!
  5. Submit your webpage below.
