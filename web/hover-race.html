Hover Race
 In this project, we're going to make a game!  Using the arrow keys to navigate the octopus,  try to center it over the other octopus to score a point.  Once you do, one of the octopuses will move to a new spot on the screen.  Try to score as many points as you can in 1 minute!

​For an example, click here. 
  1. Go to jsbin.com & login or create an account
  2. Add an appropriate page title
  3. Find a fun picture on google to use as the object and have two show up on the screen.  (They must be the exact same size.)
  4. Make one of the objects move based on the four different arrow keys
  5. When the game starts, have the object that you can't move go to a random location on your screen.
  6. If the two objects are exactly overlapped, have the non-moving object move to a new random spot
  7. Make a timer that starts at 60 seconds and counts down to 0
  8. Make a scoreboard that starts at 0 and goes up when the two images are overlapping
  9. When the timer gets to 0, have the objects disappear and have "Game Over" show up along with a button that (if clicked) refreshes the page to start a new game
  10. Save and submit your project!
