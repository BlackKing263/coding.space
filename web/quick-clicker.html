Quick Clicker

In this project, we'll make a simple game that counts how many times you can click before time runs out.

Click here to view fullscreen.

1. In our index.html, there is a sentence telling you how many clicks there have been. In index.js, use JavaScript to say they've clicked "zero times".

and display that variable's value instead of 0.

2. Now use JavaScript to create a variable to store how many times the user has clicked, and set it to the number 0. It's much easier to add to a number than change the word "zero" to "one"! Now display that variable instead of the word "zero".

3. Increase that variable's value by one every time someone clicks the click-button button in index.html. (If you haven't used event listeners yet, look them up.). Make sure to display that variable every time they click!

4. In JavaScript, make it so that when the "0" button is clicked, "0" appears on the screen.

5. Now make it so that when any number button is clicked, it gets added to the screen.

6. Do the same for every other button except equals.

7. Next to the screen, or under the keypad, create a "delete" button.

8. Make it so that when the "delete" button is clicked, one character at a time gets deleted from the screen.

9. Next to the "delete" button, make a "clear" button.

10. When the "clear" button is clicked, make it clear the whole screen.

11. When the equals button is clicked, make it do what's on the screen.

Hint: check out eval()

12. Add style to your page with CSS!
