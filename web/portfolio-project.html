Portfolio

In this project you'll make a portfolio!

1. Add a heading to the page (ex. My Portfolio, Sam's Games, etc.)

2. Under the heading, write a paragraph about your coding experience, favorite projects, and things you want to learn!

3. Using smaller headings, make two sections (one for Scratch projects and one for Web projects).

4. Using CSS, change the background color of the main heading.

5. Embed your favorite Scratch projects (you might have to open the output window in it's own window to see them)

6. Find a fun picture on the internet and embed it below your heading.

7. Center all of the content on the page.

8. Make the background color of the website your favorite color (and change the text on the page to a color that makes it still readable)

9. Continue to develop your page. Add more Scratch projects, change the fonts, add an introduction section about yourself -- you can do anything. Get creative!

10. Submit your webpage below!
