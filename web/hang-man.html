Hang Man
In this project, we're going to make a Hang Man game with different levels and One Player/Two Player modes.
​
​For an example, click here. 
  1. Go to jsbin.com & login or create an account
  2. Add an appropriate page title and heading
  3. Create 26 letter buttons
  4. Choose a word to be the answer and store it in a variable. (We will make it a random word later, but during testing phases it's easier if you know exactly what the answer is.)  
  5. Using JavaScript, create a <div> for each letter in the word. (Tip: right now you know exactly how many letters are in the word, but once it's a random word you might not.  Using word.Length() and a for loop, can you automatically create one div per letter no matter how long the word is?)
  6. Append the divs to the page.  (Tip: to get the line at the bottom, try using borderBottom.)
  7. Using a <canvas> tag, "draw" the body of the Hang Man and the stand.  (Tip: The w3schools page about canvas tags is very useful!)  
  8. When one of the letter buttons is clicked, disable it.  If it is in the word, have the letter appear on the correct line. 
  9. If the letter isn't in the word, have the first body part appear in the canvas. (If the user guesses a second wrong letter, the second body part should appear, after a third wrong letter the third body part should appear, etc.)
  10. If the user loses (all the body parts appear before they guess the word) have the rest of the letters appear in red and disable all the letter buttons.
  11. Once the user has won or lost, have a "New Game" button appear.
  12. Create an array with a bunch of words.  Instead of having your word be the answer every time, make it choose a word from the array at random. 
  13. Create a two player mode where the user can input a word that is the answer.  Let the user begin the game by selecting either one player or two player.
  14. Keep working until you're happy with your game.  You could add different levels or themes to the One Player mode, give the Hang Man fun accessories, etc.  The possibilities are endless!
  15. Save and submit your project!
