Pixel Drawing

In this project, we're going to make a website with a lot of tiny boxes that change colors when your mouse moves over them!

For an example, click here.
  1. Go to jsbin.com & login or create an account
  2. Add an appropriate page title
  3. Set the width and height of the HTML and body to 100% each
  4. In JavaScript, create 100 div elements and store them in a new array. (Hint: don't create the divs one by one. Instead, use a For loop.)
  5. Set the background color of the divs to black and set the height and width of the divs
  6. Append the divs to the page. (If the divs are all in a line, make them into a grid using inline-block)
  7. When the mouse moves over one of the divs, make it change to a different color (doesn't have to be random.) To do this, you will need to use a "forEach" loop and an event listener.
  8. Keep working on your project until you're happy with it. You could make the blocks change to random colors, add more or less divs to the page, allow the user to pick which color the divs change to, etc.
  9. Save and submit your project!
