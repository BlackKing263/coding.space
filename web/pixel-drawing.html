Pixel Drawing

In this project, we're going to make a webpage with crazy flashing colors!

Click here to view fullscreen.

1. Give your page a title.

2. Set the width and height of the HTML and body to 100% each.

3. In JavaScript, create 100 div elements and store them in a new array.

(Hint: don't create the div's one by one. Instead, use a for loop).

4. Set the background color of the div's to black and set the height and width of the div's.

5. Append the div's to the page.

(If the div's are all in a line, make them into a grid using inline-block).

6. When the mouse moves over one of the div's, make it change to a different color (doesn't have to be random.) To do this, you will need to use a forEach loop and an event listener.

7. Keep working on your project until you're happy with it. You could make the blocks change to random colors, add more or less divs to the page, allow the user to pick which color the div's change to, etc.

8. Save and submit your project!
