Calculator

In this project, we'll make a simple old-fashioned calculator that can do addition, subtraction, multiplication, division, and more.

Click here to view fullscreen.

1. Create a series of buttons for the number pad.

2. Create a space for the screen.

3. Whenever a number or operation is clicked, add that to the screen.

4. Whenever an operation is clicked, add that to the screen after the number.

5. Create a delete button and have it delete one character at a time from the screen when clicked.

6. Make the equals button works. Hint: check out eval()
