Hello Today

In this project we’ll make an interactive counter that lets you add or subtract to it.

For an example, click here

  1. Give your page a title.
  2. Add a heading that says “Hello! Today is DAY.” Put the word "DAY" into it's own HTML element within the heading. Make sure that the element that holds "DAY" has an id (Hint: use a <span>).
  3. Find a JavaScript function that can tell you a number 0-6 that represents the current day of the week (0 is Sunday, 1 is Monday, etc).
  4. Replace the “DAY” on the webpage with the actual day.
  5. Add a second line that says, “It’s currently 4:52pm” (but with the current time).
  6. Continue working until you are happy with your project.
  7. Submit your webpage below.
