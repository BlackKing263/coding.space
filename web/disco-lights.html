Disco Lights

In this project, we're going to make a webpage with crazy flashing colors!

For an example, click here
  1. Go to jsbin.com & login or create an account
  2. Add an appropriate page title
  3. Set the width and height of the HTML and body to 100% each
  4. Make an array that holds a bunch of different colors (You might need to use google to figure out how to use an array)
  5. Using a random number, write a function that changes the background color of the body to a random color from your list
  6. Make that function run every x seconds. Play around to decide how fast you want the colors to change.
  7. In JavaScript, create 100 div elements and store them in a new array. (Hint: don't create the divs one by one. Instead, use a For loop.)
  8. Append the divs to the page
  9. Set the height and width of the divs
  10. Make the function that changes the background color of the body change the background color of one of the divs in your array
  11. Now, make the function work on all of the divs in the array
  12. If your divs are all in a line, format them into a grid. (Hint: use inline-block)
  13. Continue working until you are happy with your project.
  14. Publish your website and submit it below
