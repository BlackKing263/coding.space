Giphy API Project

Objective

In this project we're going to make a small web app that takes a user query and displays the top gifs that match the search term. We'll learn about URL syntax, reading API documentation and wrestling it to do our will.

JS Bin on jsbin.com Click here for full screen
  1. Look for Giphy api documentaion.
    Hint: Many web companies have a link on their website just for developers.
  2. Can you find an example of a url that returns data from the server?
    Hint: Check out the "Code Examples" and "Sample Responses" sections.
  3. Put together a url that follows giphy api protocols. Be careful with the URL syntax.
    Limit the number of gifs in your response to 7. Use apiKey 'dc6zaTOxFJmzC'
  4. Test this url by putting it in your browser. If you see a wall of text, you've succeeded!
    This wall of text is called a Response. It is in JSON format.
  5. Add a chrome extension called jsonview . This presents the data from the server in a more intuitive way. Once you have the extension, reload your request and examine the structure of the data.
  6. It's time to start building your page.
    Make a form and define a function that captures the users entry in variable.
  7. Find a set of JavaScript functions that can make an AJAX sever request.

  9. Find a javascript function that takes a user entry and makes it into an api query or a valid uri. This value will go into your server request.
  10. Test your code. Are you sending a request to the server? Are you getting a valid response?
  11. Find a javascript algorithm to put a single result from this query and put it into your html.
    Note: One can't access the data from json directly.
    JSON.parse() is a method that makes the data available for your JavaScript to manipulate.
    Hint: consider using an <iframe> <\iframe> tag.
  12. Convert the last function to iterate over the server response and show all your results.
  13. Find a function to clear your input field so your user doesn't have to delete the previous entry before i.e. clear the page so that only the results of your recent search show.
  14. Add CSS to make everything look better organized.
