Counter
In this project we’ll make an interactive counter that can do simple addition and subtraction.

For an example, click here. 

  1. Give your page a title.
  2. Create an HTML element with “0” inside it.
  3. Using JavaScript, write code that adds 1 to the number and then updates the page.
  4. Create a button that says “Plus” and runs the code.
  5. Create a “Minus” button that does the opposite. 
  6. Add a text input box to the page. When you click “Plus” or “Minus”, update the number by the amount in the box. 
  7. Continue working until you are happy with your project.
  8. Submit your website below.
