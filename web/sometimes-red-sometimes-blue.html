Sometimes Red, Sometimes Blue
In this project, we’ll be making our own version of Sometimes Red, Sometimes Blue.

First, we’ll build the project in Scratch and then convert it into JavaScript. You’ll see how concepts translate from Scratch to JavaScript. For example, you’ll learn how Scratch's if-statement is very similar to JavaScript’s if-statement.
In Scratch
  1. Create a blue backdrop and a red backdrop.
  2. ​On flag click, change the backdrop to red.
  3. Edit the code so that on flag click the backdrop randomly changes to red or blue.
The Coding Space Project
In JS Bin
    1. Go to jsbin.com & login or create an account through Github.
     2. Add a  title and h1 to the HTML. (https://output.jsbin.com/nanesi)
     3. Use CSS to make the background-color of the <body> red. (https://output.jsbin.com/puropel)
     4. Use JavaScript to generate a random number. Store its value in a variable. 
     5.  Depending on the value of your random variable, have the background-color set to either red or blue. (https://output.jsbin.com/ridezul/3)
     6.  Try adding a third color! (https://output.jsbin.com/bicoduj)
     7. Keep working until you are happy with the project, then publish and submit your website below.
