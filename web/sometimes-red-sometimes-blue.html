Sometimes Red, Sometimes Blue

In this project, we'll be making our own version of Sometimes Red, Sometimes Blue.

First, we'll build the project in Scratch and then convert it into JavaScript. You'll see how concepts translate from Scratch to JavaScript. For example, you'll learn how Scratch's if-statement is very similar to JavaScript's if-statement.

In Scratch

1. Create a blue backdrop and a red backdrop.

2. On flag click, change the backdrop to red.

3. Edit the code so that on flag click the backdrop randomly changes to red or blue.

In JSBin

1. Use JavaScript (not CSS) to make the background color of the page red.

2. Use JavaScript to generate a random number. Store its value in a variable.
