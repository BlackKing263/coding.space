Click For Color

In this project, we're going to make a webpage whose background color is set via a blending of Red, Green, and Blue values. There will be buttons that add or subtract from the Red, Green, and Blue levels to change the background of the page.

For an example, click here.
  1. Give your page a title.
  2. Using JavaScript, make three variables (red, green, and blue) and set them all to random numbers between 0 and 255.
  3. Using those values, set the background color of the page. For example if you have Red = 100, Blue = 100, Green = 100, set the background of the page to be the color rbg(100,100,100). Click here to learn more about rgb values.
  4. Make a "+" and "-" button for each variable (6 buttons in total).
  5. When you click the button, add or subtract from the corresponding variable and update the color of the page.
  6. Save and submit your project!
